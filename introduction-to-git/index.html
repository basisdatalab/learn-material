Bagaimana kita menggunakan git?

Perintah dasar git

git init

Untuk membuat repository lokal baru, perintah ini akan membuat sebuah folder baru yang bernama ".git"

git status

Untuk melihat status dari repository

git add

Untuk menambahkan file ke dalam repository untuk di catat oleh git

git commit

Untuk menyimpan seluruh perubahan yang dibuat, jadi ketika kita melakukan penambahan file dengan git add kita bisa melakukan git commit untuk menyimpan perubahan itu

git push

Untuk menyimpan data ke dalam repository remote

git pull

Untuk mengambil data ke dalam repository remote

git checkout

Untuk pindah branch

git diff

Untuk membandingkan perubahan pada file

git merge

Untuk melakukan penggabungan antar branch

git remote

Untuk menambahkan remote repository baru

git reset

Untuk membatalkan perubahan pada repository lokal
